Projects


This personal website of mine was coded by me as my first project. It is a basic static site built using only HTML and CSS.

For this project I and two others were tasked with creating a dynamic website with three dynamic routes using a database. We choosed to use a movie ratings database. The first dynamic route was organizing the movie titles based on the first letter of the name. Each movie was a link that led to the second dynamic route. This was a page where the movie's ratings from various movie critics websites. The last dynamic route we made was listing out movie titles based on it's overall rating out of five stars. The home bar contained links depending if you wanted to see movies based on title name or by star raitings.

For this project I was tasked to work in a team to create a RESTful web server using the St. Paul crimes database. This was done using SQLite and javascript. This server was later used to create a single page application that displays a map of the crimes. Other features like filtering out types of crimes, searching for specific locations, and adding and deleting crime reports were also added. This was made possible by not only using the RESTful server previously made but the nominatim and Leaflet APIs and the VUE framework.
